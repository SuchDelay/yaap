# Dependencies

YAAP requires the following packages:

- mpv
- yt-dlp
- cava
- jp2a
- python (Python 3)

# Installation

## Arch Linux
sudo pacman -S mpv yt-dlp cava jp2a python

## Ubuntu / Debian
sudo apt install mpv cava jp2a python3 python3-pip yt-dlp

## Clone and Run

`git clone git@github.com:SuchDelay/yaap.git
cd yaap`

python3 yaap.py
