# Dependencies

YAAP requires the following packages:

- `mpv`
- `yt-dlp`
- `cava`
- `jp2a`
- `python` (Python 3)

---

# Installation

## Arch Linux
```bash
sudo pacman -S mpv yt-dlp cava jp2a python
```

## Ubuntu / Debian
```bash
sudo apt install mpv cava jp2a python3 python3-pip yt-dlp
```

---

# Clone and Run

```bash
git clone https://github.com/SuchDelay/yaap
cd yaap
python3 yaap.py
```
