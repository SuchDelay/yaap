# Dependencies

YAAP requires the following packages:

- `mpv`
- `yt-dlp`
- `cava`
- `jp2a`
- `python`

---

# Installation

## Arch Linux
```bash
sudo pacman -S mpv yt-dlp cava jp2a python
```

## Ubuntu
```bash
sudo apt install mpv cava jp2a python3 python3-pip yt-dlp
```

---

# Clone and Run

```bash
git clone https://github.com/SuchDelay/yaap.git
cd yaap
python yaap.py
```
